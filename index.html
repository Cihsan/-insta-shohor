How dose JavaScript work

JavaScript is one of most efficient a client-side scripting language it means JavaScript runs inside browsers so there is a engine in every browser like

  • Chrome uses V8 engine
  • firefox uses Spidermonkey engine
  • safari uses JavaScript Core engine

Programmers called JavaScript is an interpreted language which execute or translate in line by line.

besides that javascript is a Single-threaded, Non-blocking, Concurrent, Asynchronous programming language,

  • Thread is a smallest sequence of programming instruction which is executed
  • Single-threaded means only one task or operation performs at a single time
  • Non-block is not running slowly on the thread that means Non-block is running quickly
  • Concurrent means code is executed by more than one thread at the same time

LocalStories and SessionStories Difference

Session stories when user opens a tab in browser as key and value keeps store user data that is specified for the short time until reload the page, after reloading the page is automatically cleaned previous stored data.

Local stories when user opens a tab in browser as key and value keeps store user data that is specified not for the short time, it remains for the long time even reload the page is not cleaned previous stored data, if user not to delete browser history.
